PUPS Standard Library: sutil

INTRO:

String utilities. Some of them are taken from the string methods.

FUNCTIONS:

.stoi str => int
    Converts a string to the integer it represents.

.stof str => float
    Converts a string to the floating number it represents.

.stoc str => int
    Takes a one-character string and saves it as an integer.

.ctos int => str
    Converts a char int into a one-character string.
